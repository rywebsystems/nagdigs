Panduan Belajar WordPress Offline dengan XAMPP

1. Install XAMPP

Download XAMPP di: apachefriends.org

Setelah install, buka XAMPP Control Panel lalu klik Start pada:

  • Apache
  • MySQL

2. Download WordPress

Download di: wordpress.org/download

Extract file zip, ganti nama folder menjadi wordpress dan pindahkan ke folder:

C:\xampp\htdocs\

3. Buat Database

Buka browser dan ketik: http://localhost/phpmyadmin

Klik Database → Buat Database Baru, beri nama misalnya belajar_wp.

4. Instalasi WordPress

Buka browser dan ketik: http://localhost/wordpress

Isi data konfigurasi database:

  • Database Name: belajar_wp
  • Username: root
  • Password: (kosongkan)
  • Host: localhost

Lanjutkan hingga instalasi selesai, lalu isi nama situs, username, dan password admin.

5. Mulai Belajar

Login ke dashboard di: http://localhost/wordpress/wp-admin

Dari sini, Anda bisa:

  • Buat postingan & halaman
  • Mengganti tema (Appearance → Themes)
  • Install plugin (misalnya Elementor)
Tips: Jika ada error "port in use", pastikan tidak ada aplikasi lain yang menggunakan port 80/443 (misalnya Skype).
